# Серверний чат

## Клієнт

- авторизація по логіну та паролю
- реєстрація (ім'я, логін та пароль)
- бачити список усіх користувачів
- відправляти повідомлення користувачу
- бачити усі попередні повідомлення

## Сервер

- слухає підключення
- перевіряє авторизацію
- зберігає усі повідомлення
- приймає повідомлення дл користувачів
- віддає повіджомлення користувачам
